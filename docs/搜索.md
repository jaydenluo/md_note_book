### 🚀 使用方法

**1、基础搜索**
- 在笔记列表的搜索框中输入关键词
- 点击高级搜索图标打开高级搜索对话框

**2、快速搜索**
- 按 `Ctrl+K` (Windows) 或 `Cmd+K` (Mac) 调用快速搜索
- 支持键盘导航：↑↓ 选择，Enter 确认，Esc 关闭

**3、高级搜索选项**
- 分类筛选：限制搜索范围
- 搜索字段：仅搜索标题或内容
- 匹配模式：模糊匹配或精确匹配
- 结果排序：相关度、时间、标题

**4、搜索索引管理**
- 在设置中可以重建搜索索引
- 当搜索结果不准确时建议重建索引

- 标签搜索：在高级搜索中选择特定标签
